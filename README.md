## Description

- Development in NextJS
[SmartNewSystem Server](https://github.com/SmartNew-Plataform/server)
  
## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development runner in http:localhost:4000
$ yarn start

# watch mode 
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## Lint

```bash
# Analyze whether it was implementing good code development practices
$ yarn lint

```
